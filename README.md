## Intructions

### Install Dependencies

(Ensure that your browser supports web3 and has metamask installed)

Install [Node.js](https://nodejs.org/en/) `>= v10.16.0`.

Then run:

```sh
npm install
```

### Run Local

```sh
npm start
```

### Create Build

```sh
npm run build
```
